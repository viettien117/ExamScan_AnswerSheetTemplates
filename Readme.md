## 🇬🇧 English

The standard ExamScan multiple-choice answer sheet templates, A4 format, ready to print and use. Includes a sample answer key and a demo image so you can try the app immediately after downloading.

### 📋 Contents of `ReleaseSamples.zip`

**6 answer sheet templates** (PDF, sized for A4 paper):
- 📝 **10-question** answer sheet template
- 📝 **20-question** answer sheet template
- 📝 **30-question** answer sheet template
- 📝 **40-question** answer sheet template
- 📝 **50-question** answer sheet template
- 📝 **60-question** answer sheet template

**Answer key + demo image** (try the app right away, no printing required):
- 📊 **Sample answer key** file (`.xlsx`) — pre-filled with exam codes and specific A/B/C/D answers
- 🖼️ **Sample filled-in sheet image** (`.png`) — use as input for a test grading run

### 🚀 Try the app in 1 minute (no printing required)

1. Download `ReleaseSamples.zip` and extract it.
2. Open the **ExamScan** app on your phone, tap **"Select answer file (.xlsx)"**, then choose the sample answer key from the zip.
3. Tap **"Select image from gallery"** and choose the sample filled-in sheet image (`.png`) from the zip.
4. Once the app finishes grading, tap **"Finish and export file"** and open the resulting Excel file.

→ You've now experienced the full app workflow without printing a single sheet of paper!

### 🖨️ Printing answer sheets for real grading

1. Open the PDF that matches your exam's **question count** (e.g. `Mau_50cau.pdf` for a 50-question exam).
2. **Print on A4 paper at 100% scale** (Actual size — do NOT select "Fit to page", which shrinks the sheet and makes it harder for the app to recognize).
3. Have students fill in the bubbles with a **2B pencil** or a dark-ink ballpoint pen.
4. Open the ExamScan app, tap **"Capture sheet image"**, and grade automatically.

### 📊 Creating your own answer key file

The sample `.xlsx` file is a **reference structure** with real values already filled in:
- **First column**: exam code (132, 209, 357...)
- **Subsequent columns**: the answer for each question (A/B/C/D) in question order

Copy the sample file, replace the values with the real answers for your exam, save it, and load it into the app.

### ⚠️ Important printing notes

- Print at **100% scale** / **Actual size** — NOT "Fit to page" / "Stretch".
- Must be printed on **A4** paper (not A5/Letter/B5).
- Avoid photocopying multiple times — each copy loses detail and makes recognition harder.
- Keep all **3 orientation markers** at the corners of the sheet intact (don't trim the edges).
- Avoid shadows, creases, and smudges on the sheet.
